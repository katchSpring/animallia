
1. 팀소개 및 프로젝트 소개
   
- 프로젝트 명 :  **Animalia**
- - **"Animalia"는 라틴어로 동물을 의미**
- 팀원 : 장원녕, 이건희, 박정섭, 손영효, 진유림
- 소개
    - 한 줄 정리 :  동물에 관한 지식 공유
    - 내용 : **동물 관련 웹사이트나 커뮤니티로서  동물에 대한 다양한 주제와 관련된 콘텐츠, 동물에 대한 정보, 소식, 이야기 등을 나누는 곳**
